## Context
National and state dashboards needed reliable, near real-time data for hospital readiness and virus spread.

## Constraints
- High data volume and rapid change.
- Sensitive data handling and privacy constraints.
- Coordination across multiple stakeholders.

## What I built
- Serverless ingestion and processing using AWS services.
- Data modeling and warehousing for analytics.
- Governance and access controls for public sector requirements.

## Impact
- Improved prediction model outputs by 45% via finer-grained data ingestion.
- Supported dashboards for central and state governments.

## Stack
AWS Athena, Glue, Lambda, and serverless ingestion pipelines.
